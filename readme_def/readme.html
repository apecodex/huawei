使用教程

细心的教你怎么玩

第一步先下载,下载链接在上面

下载完之后，找到名为 "Sixuetang.zip" 的压缩包，右键解压(解压到哪儿都可以)

内容如下图:


然后把滚动条拉到最下面，找到 "思学糖.exe" 运行即可

细心的朋友可能会看见一个名为 "main.exe" 的运行文件，这个也可以运行，只是不推荐，因为 "思学糖.exe" 是处理过的，启动速度会快一些


打开之后，如果顺利(网络、版本正常)的话，点击 "确定" 就可以进入主界面了


然后来看看不顺利的情况

如果没有网络，或者网络异常，会显示如下提示:

如果你网络是正常的，你可以点击 "重试" 来重新检查网络

但有时候第一次打开也会提示 "网络异常" 点重试就可以了

现在来说一下更新问题:

如果要更新，运行后如下图:

更新很简单，重新来这个网站下载最新的安装包就好了

如果记不住网址，也可以这样，点击下载后自动打开浏览器到这个网站

如果点了没反应，就copy一下网址，到浏览器里面下载

现在就到主页面的教程了，我是不是很细心？

主页面进来后，第一步先登录，只有登录了才能继续下一步(放心，我不会记录你们的账号和密码的，我拿着也没用)

如果登录成功，会直接把你的课程列表显示出来，并且顶部有你的姓名及班级

如果登录失败，会提示你为什么会登录不了的，比如啥账号错了啊，还是密码错了啊。

接下来就是获取课程的章节了

首先要选中课程，再点击 "获取章节" 就可以得到课程的章节了

接着就是选择要代做的章节

① 先选择要代做哪些章节（鼠标滚动可以下拉）

② 选择代做课前还是课后，也可以两个一起

然后点击 "生成价格" 会提示你选择的课程及单元

同时 "生成价格" 的上面会显示需要的金额

确认无误后，点击 "支付" 会生成一个二维码，在倒计时结束前完成支付

如果选错了或者不想支付了，关掉窗口后订单将自动取消

注意: 关闭二维码窗口后，支付按钮会卡顿5秒，这是为了防止操作过快，5秒后可再次生成二维码

如果你扫了码，但又不想支付了，你也不关闭二维码页面，等倒计时结束后将自动关闭订单

支付成功后，会有弹窗提示支付成功...

在左下角有一个进度可以观察

代做完成之后，也会有弹窗提示你已完成

就这么多，不难操作吧?我这么细心的教程，应该是没问题了
